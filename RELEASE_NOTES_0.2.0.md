# 拾句 Shiju 0.2.0

拾句的首个公开发布版本。

This is the first public release of Shiju.

## 亮点 Highlights

- 面向移动端的速记面板
- 自动追加到今日日记
- 支持指定标题插入
- 支持模板创建今日日记
- 支持简洁的 Markdown 工具条
- 支持标题按钮的标题层级设置

- Mobile-first quick capture modal
- Append captures into today's daily note
- Insert under a target heading
- Create today's daily note from a template
- Compact Markdown toolbar
- Configurable heading level for the toolbar title button

## 本版包含 Included in this release

- 公开插件命名更新为 `拾句 Shiju`
- 双语 README
- 版本管理与 changelog
- 更适合 iPhone 的工具条布局
- 更适合公开发布的设置页文案

- Public plugin branding as `拾句 Shiju`
- Bilingual README
- Versioning and changelog docs
- Improved iPhone toolbar layout
- More polished public-facing settings copy
