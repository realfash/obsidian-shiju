# Shiju 0.2.1

This patch release brings the validated mobile UX improvements into the public release line.

## Highlights

- Language switch with `Auto`, `中文`, and `English`
- Better iPhone list continuation behavior
- Compatibility improvements for third-party keyboards on mobile
- Tag suggestions from existing Obsidian tags
- Fixed single-row tag suggestion area between the toolbar and the input box

## Included fixes

- Preserves unordered list markers such as `-`, `*`, and `+`
- Supports `Tab` and `Shift+Tab` indentation in the capture box
- Allows list buttons to prepend markers at the start of an existing line
- Fixes modal startup timing so the input area renders reliably

- `manifest.json`
- `main.js`
- `styles.css`
